# configuration/ruby

Provides a ruby client for handling configuration information.

## Setup

The application has the following environment variables

* `export APP_ENV=` - The APP environment

Add these to a file named `.dev.sh` (or any environment you would like)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'configuration',
    git: 'https://<GITHUB_TOKEN>@gitlab.com/sellernomics/configuration.git',
```
